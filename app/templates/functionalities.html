Functionalities

The Degree Planner has many Functionalities to help students to better plan their oath to success in attaining their Degrees. These Functionalities include:

Storage :

This Degree planner was made to cater to the University of the West Indies.Like most Degree Planner, the system has the capacity to store: User Data,and degree information such as: Majors,Minors,Faculties,Course information, and Requirements/Requisites.

Search:

The system allows the user to lookup the specific information for a course, major, minor, and programme. This allows the user to research information without generating a plan.

Generate Plan:

The system also allows the user to generate a plan based on the type of programme the user wants to enroll in, or whether the user wants to do a major,double major, major and minor, or major and double minor.The system also allows for the selection of free electives outside of their based plan.The system will also allow the user to export the plan in any of the accepted file types The system can allow importing of plans from the same accepted file types.

Gpa calculation:

The system will also be able to calculate the Gpa of the user. The system is also able to calculate the minimum number of courses and the minimum grade they would need for said courses to raise their Gpa.

AI:

Our Degree Planner also features an AI with the capabilities of recommending courses based on the user's preference. The AI can also give users limited background information on the course. The system also features a NER, which i used to identify entities from the user prompts and create a plan based on the identified entities.
